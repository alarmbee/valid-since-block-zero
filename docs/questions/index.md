# Kérdések

Az alábbiakban találhatók a validációs előírással kapcsolatban felmerülő, rövid és közérthető kérdések.  
Ezeket a kérdéseket GitHubon, pull request formájában bővítheted.

A kérdések alapján jogilag korrekt megkereső levelek (email sablonok) is készülnek, amelyeket a [Email minták](/templates/) oldalon találsz.
